Quick Start

JSDraw/HELM Editor is built on 100% Javascript, depending on two Javascript libraries: dojo toolkit and pako.js
  1. Load Pako.js (optional)
    <script type="text/javascript" src='../pako.js'></script>
  2. Load Dojo
    <script src="//ajax.googleapis.com/ajax/libs/dojo/1.10.4/dojo/dojo.js"></script>
  3. Load JSDraw.Lite
    <script type="text/javascript" src='../JSDraw/Scilligence.JSDraw2.Lite.js'></script>
  4. Load Pistoia.HELM.js
    <script type="text/javascript" src='../JSDraw/Pistoia.HELM.js'></script>
  5. Load JSDraw/WebService (optional)
    <script type='text/javascript' src='../JSDraw/Service.aspx'></script>
  6. Place a DIV as the placeholder
    <div id="placeholder"></div>
  7. Initilize the editor in scil.ready()
    <script type="text/javascript">
    var jsd = null;
    scil.ready(function () {
        var ed = new scil.helm.Editor("placeholder", { showmonomerexplorer: true, helmtoolbar: true, skin: "w8", width: 900, height: 450, ondatachange: ondatachange });
        jsd = ed.canvas;
    });
</script>
